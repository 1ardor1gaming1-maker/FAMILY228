# FAMILY228 – Официальный сайт Minecraft-сервера

Добро пожаловать в репозиторий сайта сервера **FAMILY228** – первого демократического марксистского общества в Minecraft.

## О проекте

Сайт представляет собой лендинг с информацией о сервере, его идеологии, жилищной программе, конституции (30 статей), FAQ, туториалом для новичков и техническими требованиями. Дизайн выполнен в стиле glassmorphism с поддержкой светлой и тёмной темы.

### Функционал

- Полностью статичный HTML-сайт (CSS и JavaScript встроены в один файл).
- 30 статей конституции, вшитых непосредственно в HTML.
- 15 вопросов FAQ с аккордеоном.
- Переключение светлой/тёмной темы.
- Сворачиваемое меню с кнопкой восстановления.
- Копирование IP сервера по клику.
- Адаптивная вёрстка (на мобильных устройствах меню скрывается).

## Размещение на GitHub Pages

Сайт полностью готов к размещению на GitHub Pages. Для этого:

1. Создайте репозиторий на GitHub.
2. Загрузите в корень репозитория файл `index.html` (основной сайт), а также (опционально) `_config.yml` и `README.md`.
3. В настройках репозитория (Settings → Pages) выберите ветку `main` и папку `/ (root)` в качестве источника.
4. Сайт станет доступен по адресу `https://ваш-username.github.io/имя-репозитория/`.

> **Важно:** Если вы используете кастомный домен, настройте его дополнительно.

## Локальный запуск

Просто откройте файл `index.html` в любом современном браузере. Для работы не требуется сервер или сборка.
